# 📁 Foundry Art Repository

Данный репозиторий предназначен для централизованного хранения и удобной навигации по графическим материалам (артам, токенам, портретам) для игровой платформы **Foundry VTT**.

**Главные правила:**
1. Строгое распределение изображений по тематическим папкам.
2. Человекопонятные названия файлов (никаких `img_001.png`).
---
## 🗂 Структура папок 
Создавай новые папки по мере необходимости, но придерживайся следующей логики:
```text
📦 root/
├── 🧟 monsters_full/          # Монстры | Полноразмерные изображения (портреты, арты)
├── 🃏 monsters_token/         # Монстры | Токены для боевой карты (вид сверху/круглые)
├── 🧝 classes/                # Подклассы и классы персонажей
├── 🎭 npc_portraits/          # Неигровые персонажи (портреты)
├── 🏰 locations/              # Локации, фоны, сцены
├── ✨ effects/                # Визуальные эффекты (заклинания, анимации)
└── 🧩 assets/                 # Прочие ассеты (иконки предметов, следов)
Важно:
Используй только латиницу в названиях папок.
Для пробелов используй нижнее подчеркивание _ или дефис -.
🖼 Правила именования файлов (Нейминг)
✅ Хороший пример:
orc_berserker_full.png
goblin_shaman_token.webp
fighter_battlemaster_portrait.jpg
❌ Плохой пример (избегаем):
DSC_1546.jpg
новый файл (копия) 2.png
!@#$%^&.webp


🤝 Как добавить свои изображения
Выбери нужную папку согласно типу контента.

Переименуй файл по правилам (латиница, осмысленно).
Загрузи файл через Add file → Upload files или через git push.
Напиши в описании коммита, что добавил (например: "добавил токен огненного гиганта").
📌 Рекомендации
🖼 Формат: используй .webp для токенов (меньше вес, прозрачность).
📏 Размер токена: рекомендуется 400x400 или 512x512 px (круглая обрезка).
🧠 Читаемость: представь, что через месяц ты должен быстро найти картинку "Маг огня 3 уровня". Если название говорит само за себя — ты всё сделал правильно.

📄 Лицензия
Все изображения принадлежат их законным правообладателям.
Данный репозиторий используется исключительно в личных некоммерческих игровых целях.
